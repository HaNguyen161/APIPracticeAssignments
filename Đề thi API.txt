Bài 1:(15 phút)
Tìm kiếm thông tin khách, hệ thống hỗ trợ API tìm kiếm như sau:[(host-name)]/api/v1/danh-sách?timKiem=
param timKiem hỗ trợ tìm kiếm theo Mã khách, Số điện thoại , Tên khách
Thao tác NSD
1. Step 1; Bấm chọn chức năng "Xem danh sách" và cho các tiêu chí tìm kiếm Tên khách =Nguyễn A
   Hệ thống gọi API và hiện thị Danh sách các khách thỏa mãn tiêu chí tìm kiếm (kết quả trả về 1 triệu bản ghi thỏa mãn)
2. Step 2; Bấm chọn 1 khách hàng cụ thể trong Danh sách Mã khách = 123456

Câu hỏi:
1. Với API đã cung cấp , Hệ thống gọi API tương ứng như thế nào ở Step 1 và 2 để thực hiện tính năng (ghi rõ ra nhé )
2. Bạn có đề xuất gì để cải thiện performance tương ứng tại Step 1 Và Step 2( ghi rõ ra đề xuất của bạn)

4. Chọn 1 user bất kì trong 4 user còn lại rồi xóa nó đi
5. Test xem xóa thành công chưa
